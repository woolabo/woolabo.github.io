# Apple Product Bezel

Source: [Apple Design Resources](https://developer.apple.com/design/resources/)

- `iphone-17-pro-silver-portrait.png` is Apple's iPhone 17 Pro Silver portrait product bezel.
- `iphone-17-pro-screen-mask.png` preserves the alpha mask from the `Screen` layer in Apple's matching portrait PSD.
- Official canvas: `1350 x 2760`.
- Official screen bounds: `x=72`, `y=69`, `width=1206`, `height=2622`.
