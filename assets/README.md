# e1 App Screen

The iPhone frame is built with HTML and CSS. The image inside it is loaded separately.

To update the screen preview, either:

1. Replace `app-home.png` with a new portrait screenshot using the same filename.
2. Change `appScreen` in `/site-config.js` to a different image path.

The current source image is 1179 x 2556 pixels. Use a full-screen iPhone screenshot without a device frame.
